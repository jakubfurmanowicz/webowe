Hello Angular

Krótki przewodnik po poleceniach

    Utworzene projketu:
    ng new nazwa_projketu

    Odtworzenie node_modules:
    npm install

    Uruchomienie serwera i otwarcie w przeglądarce (-o):
    ng serve -o

    Przerwanie serwera:
    Ctrcl + C

    Utworzenie komponentu
    ng generate component nazwa_komponentu
    ng g c nazwa_komponentu
